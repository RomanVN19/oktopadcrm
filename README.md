# Oktopad CRM

Открытая система CRM и управленческого учета.

## Функционал
- Сделки
- Задачи
- Роли: Руководитель, Пользователь, Бухгалтер
- Настройка схемы продаж
- Дополнительные поля
- Неограниченное количество пользователей
- Настройка ролей

## Доработка

При использовании [шаблона](https://github.com/romannep/oktopadcrmrework),
для обновления системы достаточно будет увеличить номер версии в package.json. 

## Роли

- Роль Руководитель - возможность смены пользователя в списках - 
регулируется правом на метод `put` сущности `SaleSchema`.
